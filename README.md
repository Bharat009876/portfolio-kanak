# Kanak Maru — Visa consultancy website

Professional static website for **Kanak Maru** (visa consultancy / [Travelclaps](https://www.travelclaps.com)).

## Stack

- HTML, CSS, JavaScript
- Responsive layout, dark theme

## Local preview

```bash
npm start
```

Then open the URL shown (e.g. `http://127.0.0.1:5173`).

## Deploy

Works on any static host (e.g. Vercel, Netlify, GitHub Pages). No build step required; keep `index.html` at the repository root and include the `assets/` folder.

## Assets

Place the portrait image at `assets/img/kanak.png`.
